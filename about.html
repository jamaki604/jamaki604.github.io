Projects I’ve Rocked

GUNDEX (Founder & Lead Software Engineer): I founded and built GUNDEX, a production mobile SaaS platform for Gundam Trading Card Game collectors. I designed and deployed a cross-platform Flutter app on iOS and Android, architected a scalable Supabase/PostgreSQL backend, implemented automated data ingestion pipelines, and launched a freemium subscription model. The platform has grown to 500+ users organically through community-driven development.

UI Web Builder: As the principal software developer and team lead, I created a drag-and-drop website builder enabling real-time customization of modular components. Built using React, TypeScript, Tailwind CSS, Node.js, Prisma ORM, and Supabase, with authentication and deployment handled via Vercel.

Spotify Search Engine: One of my earlier collaborative projects where I led development of a clean, user-focused search interface using the Spotify API and Flutter/Dart. This project strengthened my skills in API integration, UI design, and team collaboration.
